# Growth Screener — 系統規格

## 定位

從 S&P 500 + NASDAQ-100 每日篩選出值得深入研究的強勢中大型股，並附上新聞催化劑說明。
**研究起點工具，不是自動交易信號。**

## 入口指令

| 指令 | 用途 |
|------|------|
| `python3 main.py` | 執行當日篩選，輸出候選股票與新聞 |
| `python3 backtest.py` | 對過去 ~1 年每週執行回測，驗證策略表現 |
| `python3 -m http.server 8765` | 啟動本地伺服器，於 `http://localhost:8765` 開啟網頁閱讀器 |

## 報告結構

- 每次執行 `main.py` 產出一份 `daily-reports/YYYY-MM-DD.md`
- `daily-reports/index.json` 自動更新，記錄所有已產出日期（由新到舊）
- `index.html` 為網頁閱讀器，左側依月份收合列出所有報告，右側渲染 Markdown 內容

## 篩選流程

```
Universe（S&P500+NDX100, ~517支）
  ↓ 下載 1 年 OHLCV（yfinance）
  ↓ signals.py：技術指標篩選（4 個硬條件 + 評分排序）
  ↓ Top 20 候選
  ↓ info.py：抓基本面（營收成長、FCF、分析師評等）
  ↓ 死貓跳過濾（移除 or 標警告）
  ↓ 輸出表格 + 新聞
```

## 技術篩選條件（全部需通過）

1. 5 日累積漲幅的百分位 ≥ 85%（vs 自身過去 252 天）
2. 過去 5 天中 ≥ 3 天為正報酬（連續性確認）
3. 10 天前股價距 6 個月高點 ≤ -15%（基期夠低）
4. 收盤價 > MA20（均線突破確認）

## 評分公式

```
Score = 5D百分位 × 0.35 + 基期深度 × 0.25 + 連續天數 × 0.20 + 均線品質 × 0.20
```

## 基本面過濾邏輯

- **移除**：基本面差（營收衰退 > 10% 或 FCF 大額負值 或 Sell 評等）且股價未回到 52W 高點
- **保留＋警告 ⚠**：基本面差但股價已回到 52W 高點（市場已定價轉型）
- **正常通過**：無觸發任何死貓跳條件

## 回測設計

- 資料：2 年歷史，每 5 交易日執行一次篩選
- 衡量：+63 交易日（≈ 3 個月）後報酬 vs SPY
- 現行版本回測結果：勝率 68.1%、平均 3M +9.8%、alpha +1.2%
